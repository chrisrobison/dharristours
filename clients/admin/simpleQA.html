Simple Software QA Checklist

The following is an outline of the items to test for the Simple Software application. We will be testing from an end-user perspective signing up for a trial account, the correct setup of the application and sanity checking the newly created workspace.

  • Signup via Simple Software Website [http://www.simplesoftwaresf.com/]

    • "Free Trial" link opens signup dialog
    • Dialog rendered and functions properly
      • First field receives focus
      • All other inputs contain sample values
      • Field focus clears sample value
      • Re-focusing already completed field retains value entered
      • Validation performed on each field
