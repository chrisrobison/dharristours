HTML Tidy
Analyze, Validate and Cleanup your HTML and make it compliant

HTML Tidy is a tool that will analyze an HTML/XHTML/XML document for compliance with current standards and either correct the problem or, if the system cannot discern a fix, warn you about the problem. This tool will also "cleanup" your document, reformatting the content into a visually pleasing node indented format. HTML Tidy is careful not to make any modifications which would change the visual layout or display of the content from it's original look and feel. HTML Tidy will also cleanup your PHP or Javascript code. HTML Tidy will change your page replacing depracated or proprietary elements with the Standards Compliant equivalents (e.g. all <FONT..> tags are replaced with the CSS equivalent The only thing it cannot do is validate and cleanup content which is generated from PHP or Javascript. If you output a " <FONT COLOR=RED>...</FONT>" tag from your PHP or Javascript code, HTML Tidy will *NOT* be able to fix it.
